    # 请求的过程优化

    浏览器的网络请求基本上是 HTTP/HTTPS 请求。优化请求的过程有以下几个思路：构建工具性能调优、Gzip 文件压缩、图片优化。

    # 减少网络请求

    如果减少网络请求，就需要把东西存在本地了。本地存储有两个思路，一个是浏览器的缓存机制，另一个是离线存储技术。
